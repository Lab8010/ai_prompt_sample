# 反映すべき重要データ (2026年3月時点)

1. **体系と称号ルール**
   - 接頭辞（RH/DO/AU/EX）とレベル（百の位）の法則。
   - 称号条件: RHCSA (EX200) → RHCE (EX294) → RHCA (5つのスペシャリスト)。

2. **コストパフォーマンス比較 (トレノケート掲載価格ベース)**
   - **日額単価（税込）**: 
     - Red Hat (RH124LS): 約58,300円/日
     - Red Hat (DO180LS): 約69,300円/日
     - AWS / Microsoft: 約77,000円/日
     - Cisco: 約88,000円/日
     - VMware: 約99,000円/日
   - **RHLS Course の「全部入り」価値**: 
     - 1年間のeラーニング、100時間ラボ、本試験バウチャー、模擬試験(Exam Experience)、**無償の再試験1回**が全て含まれる。他ベンダーは受講料のみ（試験別）が一般的。

3. **実機試験突破の鉄則**
   - Copy & Paste（タイポ防止）、ドキュメント活用（man/help）、解く順番。

4. **自宅受験環境**
   - 外部モニター禁止（内蔵画面のみ）、デスク上整理（スマホ・時計・メモ禁止）、**飲み物はOK**。

5. **Exam Experience**
   - 本番同様のプロセス・難易度で受けられる模擬試験。RHLS利用者の必須ステップ。
