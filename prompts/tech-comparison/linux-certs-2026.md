# Task

- 最新の試験改定（2025-2026年）を踏まえた正確なロードマップを知りたい人

# 各資格の重要データ（2026年3月時点）
1. **LPIC (LPI)**: グローバル標準。LPIC-1 v5.0 / LPIC-2 v4.5 / LPIC-3 v3.0。
2. **LinuC (LPI-Japan)**: 日本のクラウド市場に最適化。2025/11にL3を刷新。旧L3試験は2026/11/30終了。
3. **Red Hat (RH)**: 100%実機試験。RHELに加え、Ansible、OpenShift、Podmanの製品群。
4. **Linux Foundation**: 本格OSS団体。CKA等への親和性。
