# My Prompt Library 🚀

AI（Claude / Gemini等）から最高精度の回答を引き出すための、厳選されたプロンプト集です。

## 📁 ディレクトリ構成

- **`/prompts`**: 具体的なユースケースごとのプロンプト本体。
- **`/templates`**: プロンプトを組み立てるための共通パーツ。
- **`/samples`**: AIが生成した回答のログや成果物。

## 🛠 プロンプトの共通設計指針
1. **Role**: AIの役割定義
2. **Context**: 最新の市場動向や制約（例：2026年の試験改定）
3. **Avoid Conflicts**: 対立を避け、ポジティブな選択指針を提示
4. **Scannability**: スキャナビリティを重視した構成案
